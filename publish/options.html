Selecting your publication option

The opendata.swiss portal only references and manages metadata. The primary data, as well as responsibility for publishing them, remain with the data supplier. When you publish your data, you can choose between four technical options. The following table will help you decide which option is best suited to your organization.

Please consider adding your expertise to this topic.   Topic enquiry
Number of datasets and update frequency Option for publication
small number of datasets, rarely updated Option 1 – Enter metadata manually using a web form (German)
more than 10 datasets, updated regularly Option 2 – Upload metadata manually using an XML file (German), see also DCAT-AP for Switzerland format
large number of datasets, updated frequently Option 3 – Upload metadata automatically using a harvester
spatial data already published or soon to be published on geo.admin.ch / Federal Spatial Data Infrastructure Option 4 – Publish metadata via geocat

The data publication process will depend on the option chosen. Please discuss your choice with the portal team, who will be happy to advise you on your selection.

Once you have chosen your option, you must prepare the metadata in the DCAT-AP standard.

Publish on opendata.swiss

Authorities from the Confederation, cantons and communes as well as third parties that perform tasks on behalf of the state can publish their Open Data on the opendata.swiss portal.

If you would like to add data to opendata.swiss and your agency or organisation is not yet publishing on opendata.swiss, please continue reading and contact us for assistance.

opendata.swiss is the production portal, which replaced the pilot portal ‹opendata.admin.ch› by February 2, 2016, implementing the decisions taken by the ‹Open Government Data› project organisation under the leadership of the Swiss Federal Archives SFA in the pilot phase.

opendata.swiss does not host data directly, but rather aggregates metadata about open data resources in one centralised location.

How can I publish my data on opendata.swiss?

You need to publish standardised information about your data, so-called metadata. When we say ‹data›, we mean ‹datasets› and ‹resources›.

For publishers, opendata.swiss offers the following ways to add their data to the metadata catalogue:

Use case Solution Get ready
I want to publish certain datasets, which are updated at rare intervals. I manually submit my metadata through a web page on opendata.swiss. For assistance, please contact us
I want to publish several data and/or data which are updated regularly. I manually import my metadata by uploading an XML file – in the DCAT-AP for Switzerland format (documentation see below) – on opendata.swiss. For assistance, please contact us
I want to publish a large number of datasets and/or datasets which are updated frequently. I automatically import my metadata running a metadata harvester. For assistance, please contact us
I have open geodata, already published or intended to be published via geo.admin.ch / Federal Spatial Data Infrastructure FSDI I configure corresponding metadata entries in www.geocat.ch, so it will be published automatically on opendata.swiss. For www.geocat.ch support, please contact geocat@swisstopo.ch

How can I test if I am ready?

  1. Get the example XML file.
  2. Fill the XML file for your metadata following the documentation of the DCAT-AP for Switzerland format.
  3. Send your XML-file for validation to the portal team: contact us
  4. The portal team gets back to you with feedback.

Scelta dell’opzione di pubblicazione

Il portale opendata.swiss collega e gestisce esclusivamente metadati. Questo significa che i dati primari restano presso il fornitore, che conserva anche la responsabilità della loro divulgazione.

Si prega di prendere in considerazione l'aggiunta di la tua esperienza qui.   Domanda sul tema

Per la pubblicazione su opendata.swiss esistono quattro varianti tecniche. La tabella seguente offre un orientamento per la scelta.

Numero di set di dati e intervallo di aggiornamento Opzione per la pubblicazione
Pochi set di dati, Aggiornamento sporadico Opzione 1: registrazione manuale dei metadati per mezzo del modulo web (tedesco)
Più di 10 set di dati, Aggiornamento regolare Opzione 2: caricamento manuale dei metadati tramite file XML (tedesco), vedi anche DCAT-AP for Switzerland format
Grande quantità di set di dati, Aggiornamento frequente Opzione 3: caricamento automatico dei metadati con un «raccoglitore» (harvester)
Geodati già o presto pubblicati su geo.admin.ch, l’Infrastruttura federale di dati geografici (IFDG) Opzione 4: caricamento automatico dei metadati con geocat

Il processo di pubblicazione dipende dall’opzione scelta. Prima di procedere, si prega di contattare i responsabili del portale per una consulenza.

Una volta scelta l’opzione più appropriata, occorre preparare i metadati secondo lo standard DCAT-AP.

Elabora Guida Domanda sul tema

Il manuale OGD è un ausilio per le persone e le organizzazioni che desiderano pubblicare «dati aperti» dell’amministrazione pubblica svizzera ed è basato sul modello Wiki: sono esplicitamente auspicati contributi di organizzazioni e persone con esperienza nel settore degli OGD. handbook.opendata.swiss è gestito dall’Ufficio federale di statistica.
